# Supervised Machine Learning for Credit Risk Prediction

## 🚀 Project Overview
This project focuses on leveraging Artificial Intelligence to predict credit risk. By analyzing historical financial data, we developed a system that classifies borrowers into risk categories, helping banks minimize "Non-Performing Loans" (NPL).

## 🛠️ Technical Stack
- **Languages:** Python
- **Algorithms:** Random Forest (My Focus), XGBoost, Naive Bayes, Gradient Boosting.
- **Libraries:** Scikit-learn, Pandas, Matplotlib, NumPy.

## 📊 Key Features
- **Data Preprocessing:** Handled missing values, outliers, and performed feature encoding for categorical financial data.
- **Ensemble Learning:** Utilized Random Forest to build multiple decision trees, ensuring the model is not overfitted to specific training samples.
- **Performance Evaluation:** Models were tested using Accuracy, Precision-Recall curves, and Confusion Matrices to ensure the safety of financial predictions.

## 📈 Results
The Random Forest model provided a stable and interpretable solution for risk assessment, effectively identifying high-risk individuals while maintaining a low false-positive rate.
